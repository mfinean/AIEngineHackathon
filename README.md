# AIEngineHackathon

Instructions for setup

Set Up Next.js with Vercel
```
npx create-next-app@latest rate-my-outfit
cd rate-my-outfit
npm install
npm install -g vercel
npm install lucide-react
npm install openai
npm i framer-motion
npm install playwright
npx playwright install chromium
npm install serpapi
vercel login
vercel
```
